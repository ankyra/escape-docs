Escape

Escape is a state of the art release engineering and deployment tool that can manage the full software life-cycle.

Composable Platforms

Using a few key abstractions Escape can hook into any part of a release process; whether it's managing a few files or composing a complete platform across multiple environments.
An automatically generated CI pipeline

Continuous Integration

Escape is easy to fit into existing build systems and CI pipelines, and enables best practices in versioning, testing and releasing.

Scales with your project

Escape can be run from a laptop, as a step in a CI or CD system, but also as a service that can provide on-demand, self-deploying platforms.
The Ankyra Registry

Component Registry

The Ankyra Registry contains tested Escape packages for Kubernetes, Docker, Prometheus, Etcd, and many other projects, maintained and kept up to date by Ankyra.
